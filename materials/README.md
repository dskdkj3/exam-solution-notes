## 目录结构

建议结构：

```text
materials/
  <course-slug>/
    <yyyy-mm-exam-slug>/
      README.md
      questions.md
      answers.md
      guided-solutions.md
      teaching.md
      corrections.md
      manifest.json
```

示例：

```text
materials/
  gaoshu1-ii/
    2023-06-final/
```

单套材料的 `README.md` 要直接告诉读者：

- 这是什么考试或练习；
- 哪个文件适合先看；
- PDF 下载链接在哪里；
- 是否有已知纠错或版本说明。
