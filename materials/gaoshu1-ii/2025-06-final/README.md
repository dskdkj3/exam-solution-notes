## 质量状态

- 题面由 GPT-5.5 视觉 OCR 初稿、低清原题扫描和清晰解析版交叉整理，状态为 `agent_checked_draft`，尚未声明人工逐字审定。
- 答案先独立求解，再把来源解析作为非权威材料做 reference-check；来源解析不默认正确。
- 第 11-16 题已按“一题一个独立 subagent”补跑不同解法审查；采纳路线写入参考答案和详解，过重或收益不高的路线只留在工作区记录。
- PDF 已统一为 Chrome/Skia + `print.css` 样式，带低调页眉、页脚和页码；浏览器默认路径/日期页眉页脚保持禁用。
- 第 6-10 题填空题题头按原题空格句式或题面原文片段呈现，不使用自造题型标题。
- 第 10、15 题存在清晰解析版确认的 A/B 卷差异。本目录主卷是 B 卷，A 卷只作为变体附录。
- 已记录第 16 题来源解析笔误：$r^3-1=0$ 的实根被误写为 $r_1=2$，本版按 $r_1=1$ 修正。
- v1.0 是第一版公开整理；如果你发现题面、答案或排版问题，请开 issue 并写清文件和题号。
